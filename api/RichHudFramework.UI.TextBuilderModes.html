Line breaking modes for ITextBuilder

public enum TextBuilderModes

Fields

Unlined = 1

Forces all text onto a single line. Line break characters ('\n') are filtered out or ignored.

Lined = 2

Allows text to be separated into multiple lines using manual line breaks ('\n'). Automatic wrapping is disabled.

Wrapped = 3

Text is split into multiple lines automatically based on LineWrapWidth, in addition to manual line breaks ('\n').
